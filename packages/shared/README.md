# Shared Package

`packages/shared` хранит общий слой контрактов между `apps/web` и `apps/api`.

Текущая цель пакета:
- быть единым источником transport DTO для frontend;
- фиксировать URL-контракты backend endpoints;
- уменьшать дублирование формата данных между BFF-слоем Next.js и Django API.

Ограничения текущего этапа:
- Django не импортирует TypeScript напрямую, поэтому backend пока использует эти контракты как инженерный reference, а не как runtime-зависимость;
- UI-модели не выносятся сюда, если они отличаются от transport-формы API;
- в пакет не добавляются сторонние библиотеки валидации без отдельного согласования.
