# Infra

Каталог `infra/` зарезервирован под инфраструктурные материалы проекта.

Планируемое наполнение:
- reverse proxy;
- контейнеризация;
- локальные и production-окружения;
- CI/CD-конфиги;
- инфраструктурные шаблоны запуска.

Текущее состояние:
- каталог пока остается пустым слотом и не содержит рабочей инфраструктурной реализации.
