# Web App

Frontend-приложение проекта `INFINDA` на `Next.js`.

## Команды

- `npm install`
- `npm run dev`
- `npm run lint`
- `npm run typecheck`
- `npm run build`

## Структура

- `src/app` — маршруты и layout.
- `src/features` — функциональные модули.
- `src/shared` — общие конфиги, типы и переиспользуемые сущности.

## Ограничения текущего этапа

- В приложение уже перенесена первая версия `главная`.
- `header` и `footer` вынесены в общий layout-слой для повторного использования на всех страницах.
- Реальный API еще не подключен.
- Остальные HTML-макеты пока остаются эталоном и ждут поэтапного переноса.
